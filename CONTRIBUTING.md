# Contributing to Zlija

First off, thank you for considering contributing to **Zlija**! We welcome contributions from developers of all skill levels.

## How Can I Contribute?

* Reporting bugs or edge cases in disk I/O.
* Suggesting performance optimizations.
* Improving documentation and code examples.
* Submitting Pull Requests for open issues.
